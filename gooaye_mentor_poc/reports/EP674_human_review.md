# EP674 Human Review

這份文件是給使用者的 15–30 分鐘快速 review。目標不是逐字校對 ASR，也不是檢查所有 39 個 topic segments。

請重點判斷：topic 邊界與分類結果是否保留未來投資 mentor 需要的資訊，例如投資推理、條件、風險、不確定性、例外、觀點修正、公司/產業/數字，以及講者行為傾向。

請忽略小型標點、空格或不影響語意的小錯。不同集數可能有非常不同的內容比例，所以 route 百分比奇怪不一定代表分類錯；要看段落本身是否被切對、標對、保留足夠上下文。

## How To Review

每個項目請快速填寫以下判斷：

- Topic boundary: good / starts too early / starts too late / ends too early / ends too late / should be split / should be merged with adjacent segment
- Content classification: correct / partly correct / incorrect
- Investment relevance: high / medium / low / none
- Reasoning preservation: complete / mostly complete / important context missing / reasoning split across segments
- Important information preserved: company/entity names, numbers, conditions, causal relationships, risk or uncertainty, exceptions/counterexamples, opinion change, speaker behavioral tendencies
- Note: what is wrong, what should change, whether this is acceptable for the POC
